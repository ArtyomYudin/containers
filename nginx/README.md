# Package for NGINX

* Debian bookworm based on bitnami minideb package
* latest nginx
* 
* SITES="" - set site name
* ENDPOINT="" - set backend if proxy mode 
* TZ="" set timezone
* NGINX_SKIP_SAMPLE_CERTS="false" - selfsigneg ssl cert 
* PROXY="false" - proxy or fastcgi mode
